Question 5: Difference between Questions 3 and 4

Question 3 copies a specific file named "P3" from the P3 directory to the Command directory.
Question 4 copies all files inside the P3 directory to the Command directory.

The key difference is:
- Q3: Copies one specific file (cp ../P3/P3)
- Q4: Copies all files in the directory (cp ../P3/*)
